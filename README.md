## âš–ï¸ Regulatory Hierarchy (Risk-Based Approach)
This framework utilizes the EU's four-tier risk classification to determine governance requirements:

1. **Unacceptable Risk:** (e.g., Social Scoring) -> **Action:** Immediate Decommissioning.
2. **High-Risk:** (e.g., Medical Diagnostics, HR Recruitment) -> **Action:** Strict Conformity Assessments & Logging.
3. **Limited Risk:** (e.g., Chatbots) -> **Action:** Mandatory Transparency Disclosures.
4. **Minimal Risk:** (e.g., Spam filters) -> **Action:** Voluntary Codes of Conduct.
